## Overview

Stream sensor readings from Arduino to Python over serial, then parse and visualize them in Python scripts.

## Code Explained

- Reads an analog sensor on `A0` and normalizes its value.
- Sends `RAW` and `NORM` values over serial at 115200 baud.
- Designed for easy parsing in Python.

## How To Use

- Use `pyserial` in Python to read lines from the COM port.
- Plot values with `matplotlib` or process in real-time.
